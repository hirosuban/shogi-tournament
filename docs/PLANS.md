# PLANS.md

実行プランの管理方針とアクティブプランのインデックスです。

---

## プランの種類

| 種類 | 用途 | 保存場所 |
|---|---|---|
| **軽量プラン** | 小規模な変更・単発タスク | PR の説明に記載 |
| **実行プラン** | 複雑な作業・進捗ログが必要なもの | `docs/exec-plans/active/` |

---

## 実行プランのライフサイクル

```
active/ → (完了後) → completed/
```

1. 新しい実行プランは `docs/exec-plans/active/` に作成する
2. プランには進捗ログと意思決定の記録を含める
3. 完了したプランは `docs/exec-plans/completed/` に移動する
4. 既知の技術的負債は `docs/exec-plans/tech-debt-tracker.md` に記録する

---

## 実行プランのテンプレート

```markdown
# [プラン名]

## 目標
<!-- このプランで達成したいこと -->

## 背景・動機
<!-- なぜこれが必要か -->

## アプローチ
<!-- 実装方針 -->

## タスク
- [ ] タスク1
- [ ] タスク2

## 意思決定ログ
<!-- 途中で行った設計判断を記録 -->

## 完了条件
<!-- 何をもって完了とするか -->
```
