# Phase 3: フロントエンド

## 目標

地図によるエリア選択と日付フィルタを備えた、見やすい大会スケジュール画面を実装する。

## 背景・動機

既存サイトのリスト形式では大会を探しにくい。
地図UIと日時フィルタで「行ける大会」を直感的に見つけられる体験を提供する。

## アプローチ

- 生HTML + Vanilla JS（フレームワーク不使用）で実装
- `Leaflet.js` + 都道府県GeoJSONで首都圏地図を表示
- 都道府県クリックでエリア絞り込み → APIからデータ取得 → カードリスト更新

## 意思決定ログ

- Vue.js は状態管理の複雑さがなく不要と判断。生JS + Leaflet.js で十分（詳細: `docs/design-docs/tech-stack.md`）
- 日付ピッカーは外部ライブラリ不使用・`<input type="date">` で対応
- 首都圏GeoJSONはまず静的ファイル（`src/frontend/data/shutoken.geojson`）で管理し、境界形状の高精細化は必要時に差し替える方針
